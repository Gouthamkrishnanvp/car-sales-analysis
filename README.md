# Car Sales Dashboard

## Objective
- To analyze overall car sales performance and identify revenue trends.
- To evaluate profit margins across different regions and product categories.
- To measure Year-over-Year (YoY) sales and profit growth.
- To identify Top and Bottom performing counties/regions based on revenue.
- To analyze monthly sales trends and detect seasonal patterns.
- To build interactive dashboards using Power BI for better decision-making.
- To apply DAX calculations for KPIs such as Total Sales, Total Profit, and Growth %.
- To enable dynamic filtering using slicers for deeper data exploration.

### Skills Learned

- Data Cleaning and Preprocessing using Excel
- Writing SQL queries for data extraction and analysis
- Data Modeling and Relationship Building in Power BI
- Creating Calculated Columns and Measures using DAX
- Building Interactive Dashboards and KPI Cards
- Performing Year-over-Year (YoY) Growth Analysis
- Top & Bottom Performance Analysis using filters and variables
- Identifying Sales Trends and Seasonal Patterns
- Data Visualization Best Practices
- Converting Raw Data into Actionable Business Insights

### Tools Used

- **Power BI** – Dashboard creation and data visualization  
- **Microsoft Excel** – Data cleaning and preprocessing   
- **DAX (Data Analysis Expressions)** – Creating calculated measures and KPIs  
- **Power Query** – Data transformation and modeling
